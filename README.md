# 💊 Pharmacy Management System

A Database Management System project developed using MySQL to manage pharmacy operations efficiently.
This project demonstrates database design, table relationships, SQL queries, views, triggers, procedures, and transaction management.

---

## 📌 Project Overview

The Pharmacy Management System helps manage:

* Medicines
* Customers
* Employees
* Suppliers
* Billing
* Stock Management
* Prescriptions
* Orders

This project was created as part of a DBMS academic project to demonstrate relational database concepts and SQL operations.

##  How to Run

1. Install MySQL or MySQL Workbench.

2. Create a new database:

   ```sql
   CREATE DATABASE PharmacyDB;
   ```

3. Select the database:

   ```sql
   USE PharmacyDB;
   ```

4. Open the `PharmacyDB.sql` file.

5. Execute the SQL script to create tables and insert data.

6. Run the queries included in the file to test the project.
